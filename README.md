 - React

 - Redux/toolkit

 - Chakra UI
 - Axios
 - Formik
 - Firebase
 - LocalStorage
 
# Getting Started

To get a local copy up and running follow these steps.

Clone the repo
https://github.com/KrzysKolo/contactsAPP
<br><br>
Install NPM packages <br>
<b> npm install </b> <br><br>
Start the project <br>
<b> yarn start </b>
